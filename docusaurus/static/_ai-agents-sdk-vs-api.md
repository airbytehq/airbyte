You can use Airbyte Agents programmatically three ways:

- [**SDK**](/ai-agents/interfaces/sdk): a typed Python library. Best for Python apps, notebooks, scripts, and AI agents.
- [**CLI**](/ai-agents/interfaces/cli): a shell interface. Best for scripts, CI jobs, and AI-agent harnesses that call command-line tools.
- [**API**](/ai-agents/interfaces/api): an HTTP API. Best for non-Python backends and languages the SDK doesn't cover.

Most Airbyte Agents operations are available through these interfaces. Where the SDK or CLI can't do something directly, use the API as noted explicitly in docs.
