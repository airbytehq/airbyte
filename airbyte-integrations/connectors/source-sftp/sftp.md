# SFTP

## Sync overview

The SFTP source supports Full Refresh and Incremental sync.


### Features

| Feature | Supported?\(Yes/No\)
| :--- | :--- |
| Full Refresh Sync | Yes |
| Incremental - Append Sync | Yes |
| Namespaces | No |

### Limitations

- Support CSV file type only
- All files must have the same schema

## Getting started

### Requirements

* SFTP Host
* SFTP User name
* SFTP Password
