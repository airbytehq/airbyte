# Airmeet

This page contains the setup guide for the Airmeet source connector.

## Prerequisites
- Airmeet API credentials (Access Key and Secret Key)

## Setup Guide
1. Obtain API credentials from Airmeet dashboard
2. Configure the connector with:
   - Access Key
   - Secret Key
   - Start Date (optional)

## Supported Sync Modes
| Feature | Supported? |
|---------|------------|
| Full Refresh Sync | ✅ |
| Incremental Sync | ❌ |

## Supported Streams
- airmeets
