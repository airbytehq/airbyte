# Telegram Bot API Source

This is the repository for the Telegram Bot API source connector, written in the
Airbyte Low-Code/Declarative framework.

## Local development

### Prerequisites

- A Telegram Bot token obtained from [@BotFather](https://t.me/botfather)

### Reading data

For general information on the declarative connector framework and how to
locally run, test, and contribute, see the
[Declarative Source Connector Development Guide](https://docs.airbyte.com/connector-development/config-based/low-code-cdk-overview).
