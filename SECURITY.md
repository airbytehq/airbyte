# Security policy

## Reporting a vulnerability

Airbyte takes security issues seriously. If you believe you have found a vulnerability in any Airbyte product, email [security@airbyte.io](mailto:security@airbyte.io) instead of opening a public GitHub issue.

In your report, include a clear description of the issue, affected product or component, potential impact, and steps to reproduce when possible. The Airbyte security team will review your report and follow up as soon as possible.

Use this address only for undisclosed vulnerabilities. For fixed issues, configuration questions, or help using Airbyte security features, use the [Discourse forum](https://discuss.airbyte.io/), [Community Slack](https://slack.airbyte.com/), or [Airbyte Support](https://support.airbyte.com/).

For more information about Airbyte's security practices and compliance certifications, visit the [Airbyte Trust Center](https://trust.airbyte.com/) and the [Airbyte security documentation](https://docs.airbyte.com/platform/operating-airbyte/security).
