# Jamf Pro
Mobile device management

## Configuration

| Input | Type | Description | Default Value |
|-------|------|-------------|---------------|
| `subdomain` | `string` | Subdomain. The unique subdomain for your Jamf Pro instance. |  |
| `username` | `string` | Username.  |  |
| `password` | `string` | Password.  |  |
