# Rootly
## Overview

The Rootly source supports _Full Refresh_ as well as _Incremental_ syncs.
Documentation: https://rootly.com/api#/

## Configuration

| Input | Type | Description | Default Value |
|-------|------|-------------|---------------|
| `api_key` | `string` | API Key.  |  |
| `start_date` | `string` | Start date.  |  |
