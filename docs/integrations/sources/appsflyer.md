# AppsFlyer

The Airbyte Source for [AppsFLyer](https://www.appsflyer.com/)

## Supported Streams

| Category                     | Status                |
|------------------------------|-----------------------|
| Raw Data (Non-Organic)       | ✔️ (Except Reinstall) |
| Raw Data (Organic)           | ✔️ (Except Reinstall) |
| Raw Data (Retargeting)       | ✔️                    |
| Ad Revenue                   | ❌                     |
| Postback                     | ❌                     |
| Protect360 Fraud             | ❌                     |
| Aggregate Report             | ✔️                    |
| Aggregate Retargeting Report | ✔️                    |
