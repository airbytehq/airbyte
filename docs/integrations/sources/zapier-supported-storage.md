# Zapier Supported Storage

## Overview

The Zapier Supported Storage Connector can be used to sync your [Zapier](https://store.zapier.com/) data

#### Data type mapping

| Integration Type | Airbyte Type | Notes |
|:-----------------|:-------------|:------|
| `string`         | `string`     |       |
| `integer`        | `integer`    |       |
| `array`          | `array`      |       |
| `object`         | `object`     |       |
| `boolean`        | `boolean`    |       |

### Requirements

* secret - The Storage by Zapier secret.
