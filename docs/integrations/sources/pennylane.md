# Pennylane

## Configuration

| Input | Type | Description | Default Value |
|-------|------|-------------|---------------|
| `start_time` | `string` | Start time, used for incremental syncs. No records created before that date will be synced.  |  |
| `api_key` | `string` | Pennylane API key.  |  |
