# SDK

The Airbyte Agents Python SDK lets you build, authenticate, and execute agent connectors directly in your Python applications. Together with [connectors](../../connectors/), it's the subscription-based, cloud component that manages credentials and data replication.

If you're building in a language other than Python, or calling Airbyte Agents from a non-Python backend, see the [API](../api) interface instead.

## Log in and sign up

Log in and sign up to the Airbyte Agents at [app.airbyte.ai/](https://app.airbyte.ai/).
