# Airbyte Agents SDK

The Airbyte Agents SDK lets you build, authenticate, and execute agent connectors in your own applications using the Python SDK and platform APIs. Together with [connectors](../../connectors/), it's the subscription-based, cloud component that manages credentials and data replication.

## Log in and sign up

Log in and sign up to the Airbyte Agents at [app.airbyte.ai/](https://app.airbyte.ai/).
