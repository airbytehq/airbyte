# Airbyte Agents release notes

## June 4, 2026

MCP

- Read-only MCP operations like searching connected data no longer prompt for approval before executing.
- You can now list, switch between, and query data across your workspaces through the MCP. Your active workspace selection persists across sessions.

Connectors

- The Slack connector now supports joining public channels on demand.

API

- Fixed an issue where duplicate workspace names could cause operations to target the wrong workspace.

## May 4, 2026

Initial release of Airbyte Agents.
